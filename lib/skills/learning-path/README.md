# Personalized Learning Path Generator

Creates customized learning paths for engineers based on their current skills, career goals, learning preferences, and time constraints. Generates structured curricula with phased topics, practice activities, milestones, weekly schedules, success criteria, and check-in schedules.

## What You'll Be Asked

- **Path type**: Level progression, skill acquisition, role transition, or specialization
- **Current and target role**: Where you are now and where you want to be
- **Time commitment**: Weekly hours available for learning
- **Timeline**: Target duration for achieving the goal

## Usage Examples

- "Create a learning path for a junior Angular developer who wants to become mid-level"
- "I'm a backend engineer wanting to learn Angular to become full-stack"
- "Create a learning path for a senior engineer pursuing staff level"
- "Build a study plan for learning RxJS and state management over 3 months"

## Key Features

- Skills assessment with current vs. target level gap analysis
- Phased curriculum with 2-4 week chunks for manageability
- 30/70 theory-to-practice balance in all plans
- Concrete milestones at the end of each phase
- Weekly schedule with realistic time allocation
- Check-in schedule with specific discussion topics
- Support for level progression, role transition, skill acquisition, and specialization paths

## Installation

```bash
hit install learning-path
```
