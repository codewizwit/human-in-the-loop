# Security Review

Conducts comprehensive security analysis of your codebase using automated code scanning. Covers OWASP Top 10 (2021), authentication flaws, injection attacks, insecure dependencies, and compliance violations. Provides severity ratings with CVSS scores, exploit scenarios, and remediation guidance with secure code examples.

## What You'll Be Asked

- **Review scope**: Full audit, API security, authentication, dependencies, or specific modules
- **Compliance requirements**: GDPR, PCI DSS, HIPAA, SOC 2, or none

## Usage Examples

- "Please review this codebase for security vulnerabilities"
- "Review the API routes for security issues, especially authentication and injection"
- "Analyze the authentication system for security flaws"
- "Check our dependencies for known vulnerabilities and CVEs"

## Key Features

- OWASP Top 10 (2021) coverage with CWE ID references
- CVSS severity scoring for all findings
- Step-by-step exploit scenarios for each vulnerability
- Secure code remediation examples
- Compliance assessment (GDPR, PCI DSS, HIPAA, SOC 2)
- Prioritized remediation roadmap with timelines
- Language and framework-specific security best practices

## Installation

```bash
hit install security-review
```
