# BDD Scenario Generator

Generates comprehensive behavior-driven development scenarios from user stories, requirements, or feature descriptions. Creates Gherkin syntax scenarios with step definitions and tag strategies for Cucumber, Jest-Cucumber, or Cypress-Cucumber integration.

## What You'll Be Asked

- **Feature description**: The user story, requirement, or feature to convert
- **BDD framework**: Cucumber, Jest-Cucumber, Cypress-Cucumber, or other
- **Detail level**: Minimal, standard, or comprehensive scenario coverage
- **Additional context**: Existing step definitions, system constraints (optional)

## Usage Examples

- "Generate BDD scenarios for our shopping cart feature"
- "Write Gherkin tests for user authentication with OAuth"
- "Create feature files for CRUD operations on the projects API"
- "Convert this user story into Cucumber scenarios with step definitions"
