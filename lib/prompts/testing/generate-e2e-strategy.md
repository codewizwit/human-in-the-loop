# E2E Strategy Generator

## Overview
Generates comprehensive E2E testing strategy using Playwright or Cypress.

## Variables
- `framework`: Playwright or Cypress
- `app_type`: Web app, SPA, SSR, etc.
- `critical_flows`: List of critical user flows
