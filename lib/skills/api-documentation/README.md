# API Documentation Generator

Generates comprehensive API documentation from your codebase by analyzing controllers, routes, DTOs, and schemas. Produces OpenAPI/Swagger specifications, endpoint references, request/response examples, and developer guides for both REST and GraphQL APIs.

## What You'll Be Asked

- **Documentation type**: OpenAPI spec, GraphQL schema docs, or full documentation suite
- **Scope**: All endpoints, specific module, or update to existing docs

## Usage Examples

- "Generate API documentation for our backend service"
- "Create OpenAPI 3.0 documentation for the users API endpoints"
- "Document our GraphQL API schema with examples"
- "Update the existing API docs with the new payment endpoints"

## Key Features

- Automatic endpoint discovery from controllers, routes, and resolvers
- Valid OpenAPI 3.0.3 specification generation
- Realistic request/response examples derived from DTOs
- Authentication and authorization documentation per endpoint
- Error reference tables with status codes and resolution guidance
- Per-resource documentation for complex APIs

## Installation

```bash
hit install api-documentation
```
