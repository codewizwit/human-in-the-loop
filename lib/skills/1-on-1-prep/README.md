# 1-on-1 Pre-Read Email Generator

Generates structured pre-read emails for 1-on-1 meetings that enable effective async communication between team members and their managers. Produces professional emails with three key sections: highlights and wins, current work and focus, and questions or support needed. Adapts tone and content depth based on role level and the time period covered.

## What You'll Be Asked

- **Time period**: This week, last two weeks, this month, or custom
- **Role level**: Junior, senior, tech lead, or engineering manager
- **Context source**: Project descriptions, codebase review, or both

## Usage Examples

### Senior Engineer Weekly Update

> "I need to write a pre-read for my 1-on-1 with Jordan this week"

Produces a structured email covering shipped features with metrics, project status with risks, and specific asks around priorities and career growth.

### Engineering Manager Bi-Weekly Update

> "Need a pre-read for my 1-on-1 with Chris covering the last two weeks"

Generates a manager-focused pre-read with team performance metrics, hiring updates, strategic planning progress, and escalation requests.

### Junior Developer Monthly Check-In

> "Help me write a monthly pre-read for Taylor, I'm 2 months into the role"

Creates a learning-focused pre-read highlighting accomplishments, training progress, and questions about career path and stretch assignments.

## Key Features

- Role-aware tone adjustment (junior through manager level)
- Structured three-section format for consistent communication
- Specific, actionable asks with enough context for manager preparation
- Optional codebase discovery to supplement project descriptions
- 300-500 word target for quick reading

## Installation

```bash
hit install 1-on-1-prep
```
