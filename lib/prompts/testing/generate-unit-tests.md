# Unit Test Generator

## Overview
Auto-generates unit tests for TypeScript/Angular code with comprehensive coverage.

## Variables
- `code_input`: Code to generate tests for
- `framework`: Jest or Jasmine
- `test_type`: Unit, integration, or component
