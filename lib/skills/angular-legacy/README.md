# Angular Legacy

Provides comprehensive guidance for maintaining and extending Angular applications built before Angular 16. Covers traditional patterns including NgModules, lifecycle hooks, @Input()/@Output() decorators, structural directives, RxJS-based state management, change detection strategies, and subscription cleanup.

## What You'll Be Asked

- **Work type**: Maintaining, extending, fixing, reviewing, or migration prep
- **Angular version**: Which version of Angular the project uses (12-15)
- **Focus areas**: NgModules, lifecycle hooks, I/O decorators, directives, RxJS, change detection, or testing

## Usage Examples

- "Create a new feature module with list and detail views"
- "Review this component for memory leaks in observable subscriptions"
- "Convert this component to use OnPush change detection"
- "Set up the takeUntil pattern for subscription cleanup"
