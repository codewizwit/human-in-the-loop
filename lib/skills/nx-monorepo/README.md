# Nx Monorepo Expert

Provides comprehensive guidance for building and maintaining Nx monorepos. Covers workspace structure, project boundaries, dependency constraints, computation caching, task orchestration, module federation, CI/CD optimization, and custom generators.

## What You'll Be Asked

- **Work type**: New workspace setup, restructuring, boundaries, caching, CI/CD, federation, or review
- **Frameworks**: Angular, React, NestJS, or multiple frameworks
- **Workspace size**: Number of applications and libraries (or new workspace)

## Usage Examples

- "Set up an Nx workspace for an Angular frontend with NestJS API"
- "Configure module boundaries so feature libraries cannot import from other features"
- "Our CI pipeline runs all tests on every PR - help optimize with affected commands"
- "Implement module federation for our micro-frontend architecture"
